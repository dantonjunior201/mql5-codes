## 📌 Como Usar

1. Faça o download ou clone o repositório:
   ```bash
   git clone https://github.com/dantonjunior201/mql5-codes.git
   ```
2. Copie os arquivos `.mq5` ou `.ex5` para a pasta correta do MetaTrader 5:
   - EAs → `MQL5/Experts/`
   - Indicadores → `MQL5/Indicators/`
   - Scripts → `MQL5/Scripts/`
3. Compile no **MetaEditor** e anexe ao gráfico desejado.
